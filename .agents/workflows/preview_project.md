# Preview Workflow

This workflow starts the local development server to test the project.

1. **Start Development Server**
   Start the Next.js development server:
   ```bash
   npm run dev
   ```

2. **Access Preview**
   The application will be available at [http://localhost:9002](http://localhost:9002).

3. **Verify Functionality**
   - Ensure the home page renders correctly.
   - Check the console for any hydration or runtime errors.
   - Verify that primary navigation and interactive elements are functioning.
